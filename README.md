**Q1：搬瓦工东京 VPS 是日本原生 IP 吗？**

不能。东京 CN2 GIA 套餐绑定 JPTYO_8 一个机房，不能迁移到香港、洛杉矶、大阪等其他机房。如果你需要多机房切换，应该买 CN2 GIA-E 套餐，可以在 DC6、DC9、JPOS_1、EUNL_9 等机房之间免费迁移。

搬瓦工的流量是按月重置的，超量后会限速到 1Mbps 而不是停机，下月 1 号自动恢复。如果经常超量，可以在 KiwiVM 面板里购买额外流量包，或者直接升级到更高档套餐。
